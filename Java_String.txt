1) What is String in Java? 

String is a Java Class (part of JDK) and it is not data type like int, boolean, char etc. It is a sequence of characters and enclosed with the double quotes (" ").

For example:


String txt = "TestLeaf";

2) What are the different ways to create String Object in Java?

There are two ways to create the string object, by string literal and by new keyword.

i) String as literal like:

String s = "TestLeaf";

ii) String as Object like:

String s = new String("TestLeaf");
